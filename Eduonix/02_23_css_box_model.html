CSS Box Model



Content: Images and text appear here. You can modify it using width and height properties

Padding: It clears an area around the content. It is transparent

Border: It goes around the Content and Padding. You can modify it using several properties for its style

Margin: It clears an area outside the border. It is also transparent

Box-Sizing: Added in CSS3. This property allows us to include the Padding and Border in the element's Width and Height
