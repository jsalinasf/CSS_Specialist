Overflow

If an element doesn't fit an specified area, we can use overflow to allow the user have some way to see all of the content. It has properties:

  • Visible
  • Hidden
  • Scroll
  • Auto

Visible is the default

Only works for specified BLOCK ELEMENTS with an specified Height
